== Description ==

DebugPress is an easy to use plugin implementing popup for debugging currently loaded WordPress page with support for intercepting AJAX requests.

== Installation ==
= General Requirements =
* PHP: 5.6 or newer

= WordPress Requirements =
* WordPress: 4.9 or newer

= Basic Installation =
* Upload folder `debugpress` to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Plugin settings are available under WordPress 'Settings' panel

== Frequently Asked Questions ==
= Where can I configure the plugin? =
Open the WordPress 'Settings' menu, there you will find 'DebugPress' panel.
